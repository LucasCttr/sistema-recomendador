# How to run it
1. clone it `git clone https://github.com/LucasCttr/sistema-recomendador`
2. Create a new python environment: `python3 -m venv ".venv"`
3. Install dependencies: `pip install -r requirements.txt`
4. Run it: `python3 api.py`
5. Try it with the exiting database: `curl http://localhost:8000/users/1/recommend`
